通过code换取accesstoken值（即授权获取用户信息、小程序登录）

如果是公众号，此处的code是用户同意授权后，微信跳转 redirect_uri/?code=CODE&state=STATE 链接上的code参数

如果是小程序，此处的code是通过 wx.login 获取

代码：$accesstoken = $_W['account']->getOauthInfo($code);
公众号返回结果：{"openid":"xxxxxx","session_key":"xxxxx","unionid":"xxxxx","errcode":0,"errmsg":"xxxxx"}
小程序返回结果：{"access_token":"ACCESS_TOKEN","expires_in":7200,"refresh_token":"REFRESH_TOKEN","openid":"OPENID","scope":"SCOPE","is_snapshotuser": 1,"unionid": "UNIONID"}
